Line Following

Note - this will not work until you set up motors and sensors. The path to common.h may also be different.

task main() {
    int sensorLeft, sensorRight;
  while (1) {
    // update sensor values
    sensorLeft = (SensorValue [LEFTSENSOR]);
    sensorRight = (SensorValue [RIGHTSENSOR]);

    // what does this if statement do?
    if (sensorLeft > 55 && sensorRight > 55) {
      motor[LEFTMOTOR] = 100;
      motor[RIGHTMOTOR] = 100;
    }
  }
}

Today's Task:

In the while loop, make the robot follow a line. Consider what happens when the left sensor is on black, and consider what happens when the right sensor is on black.
