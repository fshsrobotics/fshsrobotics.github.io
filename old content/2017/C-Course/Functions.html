6. Functions

6.1 Baking a Cake

A function is a group of statements that perform a task. We already have met the main() function – this is the function that the computer calls first whenever that program starts. printf() and scanf() are other functions that do things – the first prints lines into output, and the latter asks for input and puts it into a variable.

Functions are just like asking your mum to bake a cake. There are inputs – flour, eggs, milk, there is a task – the cake mixing and baking, and an output – a cake. What matters is the inputs and the output, your mum knows the recipe and you shouldn’t need to know it.

The key attributes of a function are:

  1. Functions can have input(s)

  2. Functions can output 0 or 1 value

  3. Functions can look at the values, and should do a task, like finding the maximum number, or moving a robot at a certain speed.

6.2 Defining a Function

Let’s make a max function: a function that finds the maximum of two integers.

  1. It has two inputs – x and y, which are two numbers we want to look at.

  2. It outputs one integer – the maximum, either x or y.


// Definition of the max function
int max (int x, int y) {
        int result;

        if (x > y) {
                result = x;
        }
        else {
                result = y;
        }
        return result;
}

We know a lot from the first line. int max (int x, int y) tells us that we input two numbers in the brackets, and it should return an integer, the type on the most left.

6.3 Declaring a Function

We always declare our function near the start of our program, in order for the computer to know the structure of the variable. It is much like how we declare integers and floats before we use them.


// Declaration of the max function
int max (int x, int y);

6.4 Calling a Function

We now take a look at a full code that uses a function like max().


#include <stdio.h>

// Declaration of the max function
int max (int x, int y);

// Main function
int main () {
        int num1 = 3;
        int num2 = 5;

        // Call the function, with inputs num1 and num2
        // and put the result in 'maximum'
        int maximum = max(num1, num2);

        printf("The maximum of %d and %d is %d!\n", num1, num2, maximum);

        return 0;
}

// Definition of the max function
int max (int x, int y) {
        int result;

        if (x > y) {
                result = x;
        }
        else {
                result = y;
        }

        return result;
}

Try it out! You can now call the max() function wherever you want in the code, and you’ll be certain that it will return the maximum of two numbers. You won’t need to copy paste that if statement, you only need to copy paste the one max() line.

Functions will come in very handy for making your robots move, as well as for comparing numbers.
