1.1 Countdown: Solutions

Use a loop to print a countdown from an inputted number to 0. Save it as countdown.c.

To get you started, here is some code that will count from 0 to 12:


#include <stdio.h>

int main() {
    int counter = 0;             // Start at 0
    while (counter > 12) {       // while the counter is not greater than 12
        printf("%d\n", counter); // print out the number
        counter = counter + 1;   // add 1 to the counter
    }

    return 0;
}

5. Square: Solutions

Given a size, print out a square of # characters based on that size. You'll need to have two loops - one that loops through each row, and one that loops through each cell/column.

Save your code in square.c! Make sure your indentation is good! It will make the code a lot easier to get right.

To give you a head start, what would the result of this code be?

#include <stdio.h>

int main() {
    int i = 0; // outer loop
    int j = 0; // inner loop
    int count = 0; // result
    while(i < 3) {
        j = 0;
        while (j < 2) {
            count = count + 1;
            j = j + 1; // How many times does this code get executed? two times every outer loop
        }
        i = i + 1;
    }
}
