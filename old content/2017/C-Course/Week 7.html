task main()
{
        int lightVal; // stores light value of left sensor
        while(1) {
                lightVal = SensorValue[light];
                if (lightVal > 55) { // this number may need to change
                        writeDebugStreamLine("White (%d)", lightVal);
                } 
                else {
                        writeDebugStreamLine("Black (%d)", lightVal);
                }
        }
}

RobotC Download

1. Copy all files from the USB. Save them for your friends, in case they also need it.

2. Run ROBOTCforLEGOMindstorms_456Release.exe. Accept everything.

3. Once it has finished installing, find the RobotC.exe file. For me it was in C:\Program Files (x86)\Robomatter Inc\ROBOTC Development Environment 4.X

It might be in a different place for you.

4. Run setup4.exe and note where you have saved the exe file.

5. Run the TimeStopper exe file.

5. Copy the location of RobotC.exe (e.g. the path you found in step 3) into the first box

6. Choose the date to be tomorrow (e.g. today is 23rd, put in 24th November 2017)

7. Enter "RobotC" as the shortcut name, and click on create shortcut.

8. Now, only use that shortcut to open RobotC. Do not open it normally from the Start menu.
