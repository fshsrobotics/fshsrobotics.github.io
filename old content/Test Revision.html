Functions

Functions are like asking your mum to bake a cake. There is an input, some processing (code) and an output. The input - flour, eggs, milk, chocolate. The processing - mixing, baking. The output - a cake.

Sometimes you don't need any inputs or outputs. If we want a function to print "Hello!" every time it's called, we can do:

void printHello() {
   printf("Hello!");
}
int main() {
   printHello();
   return 0;
}

If we want to, say, print a number:

void printNumber(int x) {
   printf("My number is x!");
}
int main() {
   int number = 12;
   printNumber(number);   // input is the number 12
   return 0;
}

If we want to square a number (multiply it by itself), we have an input and an output. Note that it's changed from void to int - we are now not returning nothing, we are returning an integer.

int squareNumber(int x) {
   int square;
   square = x * x; //multiply by itself
   return square;
}
int main() {
   int number = 12;
   int newNumber, otherNumber;
   newNumber = squareNumber(number);    // square 12
   otherNumber = squareNumber(5);         // we can do this too

   printf("12x12 is %i, 5x5 is %i, newNumber, otherNumber);
   return 0;
}

What would it print out above?

Notice we don't have to write 12*12 and 5*5 separately - we just write it once and we can reuse code. It makes it easier to, say, print all squares from 1 to 20.
