1. Simple Programming

1.1 Hello World

The C programming language is a widely used language that was created by Dennis Ritchie in the early 70s. It is the successor to the B language, and it’s now used for system and application programming – and a very common language to learn programming.

“Hello World” is a standard way of starting any programming language. It’s a good way of figuring out the syntax and style of how to code in that language.

// A simple C program!

#include <stdio.h>

int main () {

printf("Hello, World!\n");

return 0;

}

This program is complete – you probably can guess what it does! It has a few things:

// A single line comment. Anything after it on the same line will not be part of your code.
#include <stdio.h> Import the ‘standard input/output’ library.
It makes your program work.
int main () { … } The main function must appear in every C program. It is the first thing the computer looks for when you run the code.
printf ( … ) A string that we want to print out. Anything within double quotes is considered a string.
"Hello, World!\n" A function that lets you print something out to the screen. The ‘f’ stands for ‘formatted’.
\n Inserts a new line – makes it pretty.
return 0 0 tells the computer that the program finished without an error.

1.2 Compile the program

Copy and save the program above into a text editor (we suggest Notepad++ or Sublime Text) and save it as hello.c. All your code should have the .c file extension.

To run your program, you first need to compile it using a compiler. A compiler converts your .c code into an executable program – a file that is full of machine code, which can be read by the computer.

In our course, we use GCC for Windows and Mac.

For Mac:

Search for and open “Terminal” using Spotlight. Type in the following command and click Install:

xcode-select --install

Navigate to your .c file by using ls to list the contents of the folder, and cd to change folder. To go down one level, use cd ../

ls
cd folder

cd ../

Once you are in the same directory, type in:

gcc hello.c –Wall –o hello

(-Wall means “with all errors and warnings”, -o hello means “call the program ‘hello’”)

For Windows:

Download GCC here: https://drive.google.com/open?id=0BwVnNvJGlGV1N2FlNTcwOFZ0ZTg

Extract the whole zip folder somewhere, such as in C:/.

Go to your computer’s environment variables. This can be done by searching for ‘environment variables’ in Control Panel or in the Start menu.

Click on Environment Variables and, under System Variables, find and double-click on Path.

Click New and enter the directory path to the location you extracted your GCC in (e.g. C:\GCC). Hit OK in all the open dialog boxes.

Now, go back to your folder with your saved .c file and open cmd.exe (or type in cmd in the Explorer address bar) to open Command Prompt.

Type in:

gcc hello.c –Wall –o hello

(-Wall means “with all errors and warnings”, -o hello means “call the program ‘hello’”)

1.3 Run the program

Make sure you have:

  1. Copied and saved the code into a file called hello.c

  2. Installed GCC using the instructions above

  3. Navigated in your terminal/command prompt to the correct folder

To run the program on Mac:

./hello

To run the program on Windows:

hello

It should come up with:

Hello, World!

Congratulations, you’ve written and run your first program!
