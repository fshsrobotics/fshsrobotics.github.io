scanf

Scanf is a function like printf, but instead of outputting to the screen, it lets people input numbers, strings, etc.

#include <stdio.h>

int main() {

    int date, month, year;

    printf("What is today's date?\n");
    
    printf("Enter the date: ");
    scanf("%i", &date);
    printf("\nEnter the month: ");
    scanf("%i", &month);
    printf("\nEnter the year: ");
    scanf("%i", &year);

    printf("Today is %i/%i/%i! :D", date, month, year);

    return 0;
}

If Statements

In programming, we want to filter and make choices with the input we get from users or the environment. In Rescue, robots will take the input (e.g. a light value from 0 to 100), decide whether it is a black line or not and then send some output to the motor.

We make choices using the if statement.

Ages

if (age >= 18) {
    printf("Person can purchase alcohol");
}
else {
    printf("Person cannot purchase alcohol");
}
  • If the parameters inside the () are met, it will do the first squiggly brackets.
  • Otherwise, it will do the second squiggly brackets.
  • >= means "greater than or equal to".
  • If age >= 18, it will become 'TRUE = 1', otherwise it will become 'FALSE = 0'.
  • For example:
    • (1 + 1 > 0) is TRUE, = 1 (and goes through)
    • (2 * 5 < 4) is FALSE, = 0 (and does not go through)

Relational Operators

These operators compare one value to another.

Equal to A != B If A equals B, true, otherwise false
Not equal to A == B If A is not equal to B, true, otherwise false
Greater than A > B
Less than A < B
Greater than or equal to A >= B
Less than or equal to A <= B

Logical Operators

These are operators you can put between or in front of relations.
OR || Output is true if any one of the inputs are true
AND && Both inputs must be true for output to be true
NOT ! If input is true, output is false, and vice versa
